# Mole(墨乐) - cloud notes
 
> A GitHub-base Cloud Note, design for open & free GitHub User.

Feature in Design: 

 - GitHub as Database
 - GitHub Page as Blog / Note for Display
 - Markdown Support
 - Issues as Todo List
 - Issues as Idea List
 
Platform in Plan: 

> Build with React

 - Electron for Desktop (Mac OS, Windows, GNU/Linux)
 - Cordova for Mobile (Android, iOS, Windows Phone)
 - Web
